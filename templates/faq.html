常见问题（FAQ）

使用过程中遇到问题？先看看下面有没有答案。

账号与登录

如何注册账号？
点击顶栏「注册」，填写用户名、密码（需再确认一次）、邮箱（可选）后提交。若看不到注册入口，说明站长已关闭注册，需联系管理员开通账号。
忘记密码怎么办？
目前需联系管理员重置密码。重置后请使用新密码重新登录。
登录失败次数过多被锁定？
连续输错密码达到上限后，该 IP 会被临时锁定（默认 15 分钟）。请稍后再试，或联系管理员。
如何修改密码？
登录后进入「用户中心」→「设置」→「修改密码」，输入旧密码和新密码即可。

发帖与内容

如何发帖？
登录后进入「论坛」，填写标题和内容后提交。帖子内容上限约 50000 字，具体以站点配置为准。
如何提交代码？
登录后进入「代码」→ 填写标题、选择语言、粘贴代码后提交。他人可查看带语法高亮的代码。
如何发布文章？
登录后进入「文章」→「发布文章」，填写标题和内容，选择「草稿」或「已发布」。草稿仅自己可见。
文章 RSS 订阅地址？
访问 /articles/feed/ 可获取 RSS 2.0 订阅，或通过站点头部的 RSS 链接添加。

个人设置

如何修改个人资料？
登录后进入「用户中心」→「设置」，可修改昵称、简介、头像、隐私、通知偏好等。
如何登出其他设备？
在「用户中心」→「设置」→「账户与安全」中，可一键登出除当前设备外的所有会话。

安全与限制

密码有什么要求？
由站长配置，通常包括最小长度（默认 8 位）、是否需字母+数字、是否需特殊字符等。
API 有调用限制吗？
有。公开 API 按 IP 限流，默认每分钟 60 次，超限返回 429。健康检查接口不限流。
站点维护时能访问吗？
维护模式下普通用户无法访问，管理员可正常进入。维护结束后会自动恢复。

管理员相关

如何进入管理后台？
使用管理员账号登录后，顶栏会出现「管理」入口，点击即可进入。
站点设置在哪里？
管理后台 →「站点设置」，可配置站点名称、安全策略、注册开关、主题、SEO、维护模式等。
如何关闭注册？
管理后台 → 站点设置 → 用户 → 将「开放注册」设为「否」。

更多说明请查看 帮助 或 关于 页面。完整 FAQ 文档见项目 docs/FAQ.md。
