# 贡献指南

感谢你对 VaelorCMS 的关注！本项目采用 **AGPL 3.0** 许可证。

## 如何贡献

1. **报告问题**：在 Issue 中描述问题、复现步骤、环境信息
2. **功能建议**：提交 Issue 讨论，确认后再实现
3. **代码贡献**：Fork → 分支 → 提交 PR

## 提交规范

- 提交信息简洁明了，如：`feat: 添加 API 限流`
- 使用中文或英文均可
- 类型前缀：`feat` / `fix` / `docs` / `style` / `refactor` / `test` / `chore`

## 代码规范

- 遵循 [CODE_STYLE.md](CODE_STYLE.md)
- Python 代码通过 `python manage.py check` 无报错
- 新增功能需同步更新文档

## 许可证

贡献即表示你同意将代码以 AGPL 3.0 许可发布。
