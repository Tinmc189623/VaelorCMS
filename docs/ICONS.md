# SVG 图标库

`assets/icons/` 下的 SVG 图标，统一使用 `stroke="currentColor"`，可继承文字颜色。

## 导航与操作

| 文件 | 用途 |
|------|------|
| arrow-left | 返回、上一页 |
| arrow-right | 前进、下一页 |
| chevron-left | 左箭头 |
| chevron-right | 右箭头 |
| home | 首页 |
| forum | 论坛 |
| code | 代码分享 |
| article | 文章 |
| search | 搜索 |
| help | 帮助 / FAQ |
| cms | 关于 |
| games | 小游戏 |
| user | 用户中心 |
| admin | 管理 |
| login | 登录 |
| logout | 退出 |
| register | 注册 |

## 用法

```html
<img src="{% static 'icons/arrow-left.svg' %}" alt="" class="icon-inline">
```

CSS 类：`nav-icon`（16×16 导航）、`icon-inline`（行内 16×16）、`tile-icon`（28×28 卡片）。
