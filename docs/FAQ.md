# VaelorCMS 常见问题（FAQ）

> 使用与部署过程中的常见问题解答。

---

## 一、账号与登录

### 如何注册账号？

点击顶栏「注册」，填写用户名、密码（需再确认一次）、邮箱（可选）后提交。若看不到注册入口，说明站长已关闭注册，需联系管理员开通账号。

### 忘记密码怎么办？

目前需联系管理员在 Django 后台或数据库中重置密码。重置后请使用新密码重新登录。

### 登录失败次数过多被锁定？

连续输错密码达到上限后（默认 5 次），该 IP 会被临时锁定（默认 15 分钟）。可在管理后台「站点设置」→「安全」中调整「登录失败上限」和「锁定时长」。

### 如何修改密码？

登录后进入「用户中心」→「设置」→「修改密码」，输入旧密码和新密码即可。新密码需符合站长配置的密码策略。

## 二、发帖与内容

### 如何发帖？

登录后进入「论坛」，填写标题和内容后提交。帖子内容上限约 50000 字。

### 如何提交代码？

登录后进入「代码」，填写标题、选择语言、粘贴代码后提交。他人可查看带语法高亮的代码。

### 如何发布文章？

登录后进入「文章」→「发布文章」，填写标题和内容，选择「草稿」或「已发布」，点击「保存」或「发布文章」。草稿仅自己可见，已发布文章对所有访客可见。

### 文章 RSS 订阅地址？

访问 `/articles/feed/` 可获取 RSS 2.0 订阅，或通过站点头部的 RSS 链接添加。

## 三、自定义页面

### 什么是自定义页面？

管理员可在 Django 后台「自定义页面」中创建页面，通过 `/p/<slug>/` 访问。例如创建 slug 为 `contact` 的页面，访问 `/p/contact/` 即可。

### 如何创建自定义页面？

1. 使用管理员账号登录
2. 进入「管理」→「自定义页面」（或直接访问 `/admin/site_app/page/`）
3. 点击「添加自定义页面」
4. 填写 slug（URL 路径，如 `about-us`）、标题、内容
5. 勾选「已发布」保存
6. 若勾选「在导航栏显示」，该页面会出现在顶栏导航

### 自定义页面支持 HTML 吗？

内容支持换行（自动转为 `<br>`）。如需富文本，可安装 Markdown 等插件扩展。

## 四、个人设置

### 如何修改个人资料？

登录后进入「用户中心」→「设置」，可修改昵称、简介、头像、隐私、通知偏好等。

### 如何登出其他设备？

在「用户中心」→「设置」→「账户与安全」中，可一键登出除当前设备外的所有会话。

---

## 五、安全与限制

### 密码有什么要求？

由站长在「站点设置」→「安全」中配置，通常包括：最小长度（默认 8 位）、是否需字母+数字、是否需特殊字符。

### API 有调用限制吗？

有。公开 API 按 IP 限流，默认每分钟 60 次，超限返回 429。健康检查 `/api/v1/health/` 不限流。可在「站点设置」→「安全」中调整「API 限流」。

### 站点维护时能访问吗？

维护模式下普通用户无法访问，管理员可正常进入。在「站点设置」→「维护」中开启/关闭维护模式。

## 六、管理员相关

### 如何进入管理后台？

使用管理员账号登录后，顶栏会出现「管理」入口，点击即可进入。

### 站点设置在哪里？

管理后台 →「站点设置」，可配置站点名称、安全策略、注册开关、主题、SEO、维护模式等。

### 如何关闭注册？

管理后台 → 站点设置 → 用户 → 将「开放注册」设为「否」。

### 如何添加自定义页面到导航？

在 Django 后台编辑自定义页面，勾选「在导航栏显示」并保存。

---

## 七、部署与安装

### 如何安装 VaelorCMS？

运行 `python main.py`，若未安装会自动进入安装向导。按步骤完成数据库配置、站点配置即可。详见 [INSTALL.md](INSTALL.md)。

### 生产环境如何启动？

使用 `python main.py --prod` 或配置 Gunicorn + Nginx。详见 [DEPLOYMENT.md](DEPLOYMENT.md)。

### 如何备份数据？

定期备份 MySQL 数据库及 `config/config.ini`、`storage/uploads/` 目录。
